VitalScan-AI

AI-Powered Health Risk Prediction Platform

A full-stack health intelligence platform that uses machine learning to predict disease risk from patient vitals — making preventive care proactive and data-driven.

The Problem

Millions of preventable health conditions go undetected until symptoms worsen, due to lack of accessible early-warning tools for everyday users.

The Solution

A web application that accepts patient vitals, runs them through a trained ML model, and surfaces risk assessments with actionable health insights — instantly, in the browser.

Key Features
ML-based disease risk prediction Patient intake form with validation Real-time confidence scores Dark-themed medical UI Deployed on Vercel

Job Recommendation System

AI-Driven Career Matching Engine

An intelligent recommendation engine that matches candidate profiles to relevant opportunities using ML algorithms — reducing noise and improving career search efficiency.

The Problem

Job seekers waste hours sifting through irrelevant listings. Traditional keyword search fails to capture nuanced skill alignment between candidate and role.

The Solution

A recommendation system that builds a semantic profile of a candidate's skills, then scores and ranks job listings using similarity algorithms — surfacing only high-relevance opportunities.

Key Features
Skill-based candidate profiling ML-powered job-match scoring Ranked recommendation feed Full-stack REST API architecture Modular ML pipeline
